USE

Click edit to set the matrix gallery. Add, edit, or delete the images you want to display the gallery. You can see here the options you have for image:

  • Image. You can enter the url of the images manually or press the button "change image". Pressing the button will load the wordpress media library, select the image you want and click "Insert in Post" to insert the image in the matrix.
  • Title. The title of each image/video.
  • Url. Image link.
  • Description. Description text.
  • SEO. Seo labels.

After the image list, the matrix has the configuration options and overall design of the matrix carousel. These are among others: width of the matrix, the matrix height, number of images rows, the images sizes, types of animation, colors, ligthbox, text, ...
